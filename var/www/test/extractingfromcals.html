Currently this tool only extracts event information from the default calendar owned by the 'BSUCAdmin ForBSUCMultiusers' Google account we administer. In the embedded calendars shown here, this calendar is rather confusingly referred to as 'shaftesburyuc@gmail.com' rather than default and would typically be the one containing the same kind of event information currently shown in Notices and Clanger documents.

The tool extracts event information between two user specified dates.

In order for the tool to run to completion, the user running it either has to sign in to a Google account during the process, or be signed into a Google account before the tool runs. Any Google account will do. This is probably because the tool runs on Google servers and they want some element of control.

If you have access to a your own Google account then it's highly recommended to use that. If you don't, then in theory it's possible to login to the 'BSUCShared MultiuserAccount' Google account that owns the calendar we are trying to extract event info from. But think very carefully beforehand about the implications of logging into a shared Google account.
