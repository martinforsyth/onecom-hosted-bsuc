Google calendar event extraction tool

The tool extracts event information between two user specified dates.

Currently it only extracts event information from the default calendar owned by the 'BSUCAdmin ForBSUCMultiusers' Google account we administer. This calendar is confusingly referred to by Google as 'shaftesburyuc@gmail.com' rather than default.

In order for the tool to run to completion, the user has to sign in to a Google account, either before running it or in response to a prompt during the process. Any Google account will do. This is probably because the tool runs on Google servers and they want some element of control.

If you have access to a your own Google account then it's highly recommended to use that. If you don't, then in theory it's possible to sign in to the 'BSUCShared MultiuserAccount' Google account that owns the calendar the tool is extracting event info from. But think very carefully beforehand about the implications of signing into a shared Google account.
